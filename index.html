INTRODUÇÃO

Os semáforos são cruciais para a organização do trânsito em cidades, pois eles garantem a segurança de veículos e pedestres. Com os avanços tecnológicos, a automação desses sistemas atualmente é mais fácil de usar e versátil, especialmente com microcontroladores como o uso do Arduino. Este artigo vai explicar como criar um semáforo usando o Arduino, desde a montagem do circuito até programar e operar o sistema.

OBJETIVOS

O objetivo deste projeto é construir um semáforo utilizando um Arduino, com o intuito de: Demonstrar o funcionamento básico de um semáforo de trânsito. Desenvolver habilidades em eletrônica básica e programação em Arduino. Aplicar conceitos de controle de tempo e sequenciamento de sinais luminosos.

MATERIAIS UTILIZADOS

Para a construção do semáforo, foram utilizados os seguintes materiais:

1 x Arduino Uno

1 x Protoboard

3 x LEDs (vermelho, amarelo e verde)

3 x Resistores de 150Ω

Jumpers (fios de conexão)

Cabo USB para conexão com o Arduino

Computador com software Arduino IDE instalado

CÓDIGO FONTE 


    int vermelho = 8;
    int amarelo = 9;
    int verde = 10;

  void setup(){

  pinMode(vermelho, OUTPUT);
  pinMode(amarelo, OUTPUT);
  pinMode(verde, OUTPUT);
}

void loop(){
  
  digitalWrite(vermelho, HIGH);
  digitalWrite(amarelo, LOW);
  digitalWrite(verde, LOW);
  delay(5000);
    
  digitalWrite(vermelho, LOW);
  digitalWrite(amarelo, LOW);
  digitalWrite(verde, HIGH);
  delay(3000);
      
  digitalWrite(vermelho, LOW);
  digitalWrite(amarelo, HIGH);
  digitalWrite(verde, LOW);
  delay(1000);

}

RESULTADOS

O semáforo juntamente com Arduino alcançou os objetivos propostos e mostrou como utilizar um microcontrolador para poder automatizar sinais de trânsito. O projeto obteve sucesso em ensinar eletrônica e programação básica e mostrou uma flexibilidade do Arduino em aplicações de controle.

DISCUSSÃO

Os resultados indicam que o LED vermelho acendeu consistentemente por 5 segundos, seguido pelo verde por 3 segundos e, finalmente, o amarelo por 1 segundo, confirmando a operação correta do circuito como um semáforo básico. Esses resultados estão de acordo com a expectativa inicial, validando a hipótese de que o circuito funcionaria conforme projetado. Uma possível limitação observada durante o experimento foi a precisão do tempo de atraso. A função delay() pode não ser precisa o suficiente para aplicações em tempo real, especialmente em sistemas mais complexos. Um método mais preciso para controlar o tempo poderia ser a utilização de temporizadores de hardware. Apesar dessa limitação, o circuito demonstrou seu funcionamento básico como esperado. Para estudos futuros, seria interessante explorar o uso de outros componentes, como sensores de proximidade para simular a detecção de veículos, ou implementar temporizadores mais precisos para maior confiabilidade em sistemas reais de semáforos. Essa discussão ajuda a contextualizar seus resultados dentro de um quadro mais amplo, mostrando o que foi aprendido e o que poderia ser melhorado no futuro.

CONCLUSÃO

O desenvolvimento do semáforo com Arduino alcançou os objetivos propostos, mostrando como um microcontrolador pode ser usado para automatizar sinais de trânsito. A execução do projeto proporcionou um aprendizado prático sobre eletrônica e programação, além de destacar a flexibilidade do Arduino em aplicações de controle.
